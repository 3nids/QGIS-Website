Quantum GIS is open source software available under the terms of the GNU General Public License meaning that its source code can be downloaded through tarballs or the git repository.

Datasets: For testing and learning purposes, a sample dataset is available, which contains collections of data from different sources and in different formats

Warning: Development versions from master are provided for some platforms in addition to the released version. 'Master' contains unreleased software that is currently being worked on. As such they are unsupported and only provided for testing purposes to early adopters to check if bugs have been resolved and that no new bugs have been introduced. At any given time it may not work, or may do bad things to your data. Use at your own risk.
